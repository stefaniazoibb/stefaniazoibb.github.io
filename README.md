# stefaniazoingb.github.io

Personal website of **Stefania Zoi** — Marie Skłodowska-Curie PhD Fellow in Computational Neuroscience & Machine Learning at Sorbonne Université, Institut de la Vision (Paris).


## Deploy
1. Push this repo to `https://github.com/stefaniazoingb/stefaniazoingb.github.io`
2. In **Settings → Pages**, set Source = `Deploy from a branch`, Branch = `main` / root.
3. The site goes live at **https://stefaniazoingb.github.io**

## Edit
Everything lives in `index.html` — copy, sections, and styling are all in that one file.
