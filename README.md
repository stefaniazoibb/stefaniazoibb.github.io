# stefaniazoibb.github.io

Personal website of **Stefania Zoi** — Marie Skłodowska-Curie PhD Fellow in Computational Neuroscience & Machine Learning at Sorbonne Université, Institut de la Vision (Paris).

A single self-contained `index.html` (no build step, no dependencies), served via **GitHub Pages**.

**Live at:** https://stefaniazoibb.github.io

## Deploy
The site is already published from this repo via GitHub Pages (Source = `Deploy from a branch`, Branch = `main` / root). Pushing to `main` redeploys it automatically.

## Edit
Everything lives in `index.html` — copy, sections, and styling are all in that one file. The portrait is `assets/me.jpg`.
